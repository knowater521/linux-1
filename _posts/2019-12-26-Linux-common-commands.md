## 3. 常用实例

1. 复制

   ```bash
   cp -r file/. .   # 复制 file 目录下的所有文件到当前目录
   ```

2. 删除

   ```bash
   rm ./*          # 删除当下所有文件，不包括目录
   rm -rf dir      # 直接删除目录和目录下的所有文件
   ```
